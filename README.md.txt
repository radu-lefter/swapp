# SwApp

Swapp is an application that lets you view and compare Swadesh Lists

## Installation

Download the repository

Install the dependencies

```bash
npm install
```
Run the seeder

```bash
node seeder
```

Run the project in development mode

```bash
npm run dev
```

## Usage

Just use as a nomal web page
